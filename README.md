# transistor-tester-m12864

Resources for the M12864 DIY transistor tester kit — an ATMEGA328-based component tester that measures
transistors, diodes, resistors, capacitors, and inductors on a little LCD.

- **[Build guide](guide.md)** — step-by-step assembly walkthrough with photos.
- **[Case](case/README.md)** — 3D-printable enclosure (OpenSCAD).
